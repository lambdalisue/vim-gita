#
# DEFAULT STATUS MAPPING
#
# <C-l> : Update status
# cc    : Switch the buffer to a commit buffer
# cC    : Switch the buffer to a new commit buffer
# cA    : Switch the buffer to a new commit buffer in AMEND mode
# 
# ee    : Edit a working tree version of a selected file
# eh    : Edit a working tree version of a selected file horizontally
# ev    : Edit a working tree version of a selected file vertically
# eE    : Edit a working tree version of a selected file vertically
# EE    : Edit a working tree version of a selected file vertically
#
# oo    : Open a exact version of a selected file
# oh    : Open a exact version of a selected file horizontally
# ov    : Open a exact version of a selected file vertically
# oO    : Open a exact version of a selected file vertically
# OO    : Open a exact version of a selected file vertically
#
# dd    : Open a diff file of a selected file
# dh    : Open a diff file of a selected file horizontally
# dv    : Open a diff file of a selected file vertically
# dD    : Compare a difference of a selected file
# dH    : Compare a difference of a selected file horizontally
# dV    : Compare a difference of a selected file vertically
# DD    : Compare a difference of a selected file
# DH    : Compare a difference of a selected file horizontally
# DV    : Compare a difference of a selected file vertically
#
# ss    : Solve conflict of a selected file (MERGE, REMOTE vertically)
# sh    : Solve conflict of a selected file (MERGE, REMOTE horizontally)
# sv    : Solve conflict of a selected file (MERGE, REMOTE vertically)
# sS    : Solve conflict of a selected file (LOCAL, MERGE, REMOTE vertically)
# sH    : Solve conflict of a selected file (LOCAL, MERGE, REMOTE horizontally)
# sV    : Solve conflict of a selected file (LOCAL, MERGE, REMOTE vertically)
# SS    : Solve conflict of a selected file (LOCAL, MERGE, REMOTE vertically)
# SH    : Solve conflict of a selected file (LOCAL, MERGE, REMOTE horizontally)
# SV    : Solve conflict of a selected file (LOCAL, MERGE, REMOTE vertically)
#
# <<    : Stage changes of a selected file
# >>    : Unstage changes of a selected file
# --    : Stage/Unstage (Toggle) changes of a selected file
# ==    : Discard changes of a selected file on working tree
#
# -a    : Add a selected file (-A to force)
# -r    : Reset a selected file
# -d    : Remove a selected file (-D to force)
# -c    : Checkout a specified version of a selected file (-C to force)
# -o    : Checkout a ours version of a selected file (for conflicted)
# -t    : Checkout a theirs version of a selected file (for conflicted)
#
